# Patrotur Turismo

Site institucional da Patrotur Turismo, agência de viagens de Patrocínio, MG.

**Site público:** [renanrigoni.github.io/patrotur](https://renanrigoni.github.io/patrotur/)

## Desenvolvimento local

```bash
npm ci
npm run dev
```

## Verificações

```bash
npm test
npm run lint
npm run build
```

O deploy para o GitHub Pages é realizado automaticamente pelo GitHub Actions a cada atualização da branch `main`.
